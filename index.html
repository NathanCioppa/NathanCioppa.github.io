Spotify AutoQueue

Spotify AutoQueue is an Android app that uses Spotify's API to automatically add songs into your queue, so that you can experience the same smooth transitions of an album while shuffling any playlist.

I came up with this idea while I was initially messing with the Spotify Web API. Later I started working on the first version of this app using React Native, because up until that point I had only known Javascript. Eventually I had it working exactly as expected, except for one thing: I was not able to keep the app running in the background, which was essential for the app to be usable. After some research, I realized that the only good way to add that functionality was to use native code. So naturally I decided to take the time to learn Java, and completely re-create the app nativley for Android :)

Now I use this app every day, and it is by far the project I am most proud of.

JoJos Mod

JoJos Mod is mod for the game Terraria, created using tModLoader— a program used to easily play and create mods for Terraria. Most of this mod's content references other nerd things that I like.

I love Terraria, and I also like modded Terraria. So naturally I became curious about how to make mods. The game is coded in C#, but I was able to start modding just fine without trying to learn it first— it was similar enough to Java (except for the fact that it's not Java so it's automatically better).

Creating this mod gave me a bit of an insight on how video games are made. One thing I realized is that it's a lot of math, which is fun in some cases, and a bit of a headache in others. I noticed that taking on projects such as this— and even just coding in general —made me significantly better at math.

I created my own wiki-type page for this mod, to document all of the items and what they reference.
